
# Kalkulator temperatur

Aplikacja webowa służąca do konwersji temperatur na inne jednostki.

# Wymagania
- PHP 7
- Baza danych SQL (Testowane na mysql)

# Konfiguracja
- consts.php
```
const DB_HOST = "localhost";
const DB_USER = "calculator";
const DB_PASS = "";
const DB_NAME = "calculator";
```
